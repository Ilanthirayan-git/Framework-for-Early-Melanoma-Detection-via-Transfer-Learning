##  Problem Statement

Melanoma is the most dangerous form of skin cancer, responsible for the majority of skin cancer-related deaths worldwide. Early detection is critical — when caught in the early stages, the 5-year survival rate exceeds 98%, but it drops to below 25% in late-stage diagnoses.

Traditional visual diagnosis by dermatologists is subjective, time-consuming, and prone to human error. Existing computer-aided diagnosis (CAD) systems often rely on single-model architectures that suffer from limitations such as:

- Low sensitivity in detecting atypical lesions
- High false-negative rates leading to missed diagnoses
- Poor generalization across diverse skin types and image qualities

**This project addresses these challenges by proposing a hybrid deep learning framework** that leverages the complementary strengths of ResNet50 (residual skip connections for deep feature extraction) and DenseNet121 (dense connectivity for feature reuse), combined with aggressive data augmentation and transfer learning — achieving superior accuracy and robustness in binary melanoma classification.

---

##  Objectives

- Build a hybrid CNN model using ResNet50 and DenseNet121 via transfer learning
- Train and evaluate on a curated, augmented melanoma image dataset
- Deploy the model as an interactive web application using Flask
- Provide real-time prediction (Melanoma Affected / Healthy) with a confidence score
- Visualize model performance using confusion matrix, precision, recall, and F1-score

##  Model Architecture

The model uses **DenseNet121** as the primary backbone with **transfer learning** from ImageNet weights:

| Layer | Details |
|---|---|
| Base Model | DenseNet121 (pretrained on ImageNet) |
| Frozen Layers | First 100 layers frozen for fine-tuning |
| Global Average Pooling | Reduces spatial dimensions |
| Dropout | 0.5 rate to prevent overfitting |
| Output Layer | Dense(1, activation='sigmoid') — Binary classification |
| Optimizer | Adam (lr = 1e-4) |
| Loss Function | Binary Cross-Entropy |

**Data Augmentation applied during training:**
- Rotation: ±40°
- Width/Height shift: 40%
- Shear and Zoom: 40%
- Horizontal & Vertical flips
- Fill mode: Nearest

##  Web Application

The Flask web app allows users to:
1. Upload a skin lesion image
2. Receive an instant prediction — **Melanoma Affected** or **Healthy**
3. View the confusion matrix for the current prediction

### Web App Flow
```
User Uploads Image → Preprocessed (224×224, normalized) → Model Inference → 
Threshold (≥ 0.5 = Healthy, < 0.5 = Melanoma) → Result + Confusion Matrix Displayed
```

##  Getting Started

### Prerequisites

- Python 3.8 or above
- pip (Python package manager)
- Git

### 1. Clone the Repository

```bash
git clone https://github.com/<your-github-username>/melanoma-detection-resnet-densenet.git
cd melanoma-detection-resnet-densenet
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

**Required packages:**
```
tensorflow>=2.8.0
flask
numpy
scikit-learn
matplotlib
seaborn
Pillow
```

### 3. Download / Place the Dataset

Place your dataset in the following structure:
```
Dataset/
├── train/
│   ├── Melanoma/
│   └── Healthy/
└── test/
    ├── Melanoma/
    └── Healthy/
```

### 4. Train the Model (Optional — Pre-trained model included)

```bash
python disease.py
```

This trains the DenseNet121 model and saves `poultry_disease_detection_densenet.h5`.

### 5. Evaluate the Model

```bash
python accuracy.py
```

Outputs: Accuracy, Precision, Recall, F1-Score, and Confusion Matrix.

### 6. Run the Web Application

```bash
python app.py
```

Open your browser and navigate to: **http://127.0.0.1:5000/**

### 7. Test a Single Image (CLI)

```bash
python test.py
```

Edit the `image_path` variable in `test.py` to point to your test image.
